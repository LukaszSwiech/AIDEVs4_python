# AIDEVs4_python

Python re-implementation of all 25 tasks from the **AIDEVs4** course
(originally completed in JavaScript). Each task builds an LLM agent;
this repo is a from-scratch redo in Python to deepen both the agent
concepts and the language.

> **The code is written by hand — not generated by AI.** AI is used only
> to evaluate solutions and for code review. The implementation is my own;
> the goal is to learn Python and agent design by actually building it.

## Techniques used

- **Agentic pipelines** — Function Calling, MCP (Model Context Protocol),
  and Structured Output (JSON Schema), with prompt caching, multi-provider
  routing, and deterministic code–LLM hybrid logic.
- **Context engineering** — prompt-cache optimization, dynamic context
  injection, and token-budget management to control cost and latency.
- **Multi-agent orchestration** — DAG-based schedulers, actor/subagent
  delegation, and event-driven agent communication.

## Models

Picked per task for the right cost / latency / capability trade-off —
from frontier models down to smaller and open ones:

- OpenAI (GPT)
- Anthropic (Claude)
- Google Gemini
- Nvidia Nemotron
- Zhipu GLM
- Alibaba Qwen
- SpeakLeash Bielik (Polish)
